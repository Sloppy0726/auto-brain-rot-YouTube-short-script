# Background Music

Put low-volume background tracks in folders that match the channel niche or fiction genre.

Suggested folders:

- `default/`
- `micro-horror/`
- `sci-fi-ai/`
- `moral-dilemma/`
- `workplace-drama/`
- `relationship-drama/`
- `scams/`
- `business/`
- `ai/`
- `internet/`
- `law/`
- `money/`

Use `--music-dir assets/music/<folder>` to choose from one folder, or `--music-dir assets/music` to choose from every folder.

Keep a note of each track's source and license. For YouTube, the safest starting point is the official YouTube Audio Library in YouTube Studio.
